## 1. 엘리베이터 구현
- 엘리베이터가 위, 아래로 움직이면서 승객을 태웁니다.

- 엘리베이터는 항상 맨 윗층과 맨 아랫층을 반복해서 움직입니다.

- 엘리베이터가 올라가는 중이면 올라가는 승객만 태우고, 내려가는 중이라면 내려가는 승객만 태우게 됩니다.

- call한 고객이 없더라도 엘리베이터는 항상 위, 아래로 움직이기 때문에 완벽한 효율은 기대하기 어렵습니다.
